# Wrapped Bullets

## Policy Capsule

- A bullet whose prose runs past the line-width limit and therefore wraps
  onto an indented continuation line.
- A second bullet that also wraps, because markdown authors reflow prose to
  the same width the rest of the document uses.
- A third bullet, single-line.
- A fourth bullet that wraps twice: once here,
  and again here, still part of the same bullet.
- A fifth bullet, single-line.

## Some Other Section

Prose that must not be counted as a capsule bullet.

- This bullet lives under a different heading and is out of the capsule.
